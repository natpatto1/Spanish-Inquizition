"Nobody expects the Spanish InQuizition!"

- Monty Python

Note that we recommend recommend you access this application using Chrome or Firefox web browser.

The Spanish InQuizition is broken down into topics. To progress to the end of the course you need to complete topics by earning points in topic games. On the homepage you can see the topics that are available to you. If a topic button is disabled (you can’t select it), this means the topic hasn’t yet been unlocked. The bars around the topic icons (red, orange and/or green) are the topic strength, this tells you how well you know and practice the topic. Try to maintain good topic strength in all your topics. You can also see how many points you have earnt over the whole course and how many points you need to earn from which topic to progress:


You can also earn points to progress by reviewing all topic vocabulary, to do this select the Reviewing All Topics button.


Go ahead and select the first topic, ‘The Basics’. This will take you to the first topic page (If you ever want to return to the homepage or select the ‘Select a different Topic’ link). On the topic page you can see a full list of all the vocabulary included in the topic:










You can see the Spanish, it’s English translation, you can listen to the spoken Spanish by selecting the ‘listen’ link. You can also see when the vocabulary is next expected to be reviewed. Keep an eye on this, if many items are overdue (have negative day values) the topic strength will become weaker. If there is any extra information about the vocabulary there will also be an i information button to view this.

The topic page is also where you can find the games to earn points and progress! There is a flashcard game and a construct game.

Launch a game session (10 questions) by clicking an icon link. For a flashcard question you will be presented with the English and 4 Spanish options:


Try to select the correct Spanish translation of the English. For every correct answer you will be awarded 1 point.



For a construct question you will also be presented with the English. If this is a single word you will also be given the letters that make up the Spanish translation as buttons. Select these or type the answer into the text field. Select the Submit button when you are happy with your answer:





If the presented English is more than one word you will be given the words that make up the phrase or sentence as buttons. But be careful (¡tener cuidado!) there will also be some extra words to distract you! Again, you can select these buttons or type the phrase or sentence into the text field and select the Submit button when you are happy with your answer (and don’t worry too much about punctuation):



For every correct construct answer you will be awarded 2 points, it’s more difficult but worth it!


If you answer incorrectly or run out of time you will lose a life (the guys with the hats are your lives). If you lose 3 lives, it’s game over and you will win 0 points!





If you make it through all 10 questions the results page will show you how you did and how many points earnt:

Each item you answered will be shown, if you answered correctly it will be green and incorrectly red. There is also a scale of correctness:

  • 5 = Perfection :)
  • 4 = Almost perfect
  • 3 = Passable
  • 2 = You’ve made a few mistakes
  • 1 = You need to practice more!
  • 0 = You didn’t even try :(

You can play another game by selecting the Play again link or go back to the topic page by selecting the Back to Topic Menu link.

Try to play at least 1 successful game a day and you can maintain a streak! You can see your current streak and longest streak, in days, on the ‘Your Activity Streak’ page which is accessible via the header bar drop-down menu:

And you can also see a calendar of all the days you have successfully completed at least 1 game.

For anyone competitive out there, you can also see how you rank compared to other players on the ‘Scoreboard’ page, accessible via the header bar drop-down menu.


You’re ready to start practicing! Please let us know how you get on by contacting us on our twitter account.
